# 📊 Middle East Dry Fruits Co. – Sales Analytics Dashboard

This repository contains a Power BI dashboard project developed during a hands-on workshop focused on analyzing sales data for a fictional dry fruits company operating across global regions.

---

## 🧠 Project Summary

The goal of this project was to transform raw sales and profit data into actionable insights. Using Power BI, the data was cleaned, modeled, and visualized to help management make informed decisions based on trends, top-performing products, and regions.

---

## 🛠️ Tools & Skills Used

- Power BI Desktop
- Power Query (ETL)
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Dashboards

---

## 📁 Files in this Repository

| File Name                              | Description                                    |

---

## 📊 Dashboard Features

- **Total Sales**: 30.3M  
- **Total Profit**: 23.3M  
- **Sales by Month**: Trend analysis of monthly performance  
- **Sales by Region**: APAC region contributes the highest share  
- **Sales by Geography**: Top countries include USA, India, and UK  
- **Sales by Product**: Top-selling products include Prunes and Apple Rings  
- **Sales by Salesperson**: Siddharth Singh was the highest performer  

---

## 🔍 Key Visuals & Insights

- **Line Chart**: Total Sale by Month to identify sales trends  
- **Bar Chart**: Total Sale by Geography  
- **Donut Chart**: Total Sale by Region  
- **Pie Chart**: Product-wise sales contribution  
- **Table**: Salesperson performance ranking  

---

## 🎯 Business Impact

This dashboard enables business stakeholders to:
- Monitor sales performance monthly and geographically
- Identify high-performing products and staff
- Allocate resources effectively to optimize revenue and profit
